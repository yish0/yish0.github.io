# Getting Started with Astro

Astro는 콘텐츠 중심 웹사이트를 만들기 위한 현대적인 웹 프레임워크입니다.

## 왜 Astro인가?

1. **제로 JavaScript** - 기본적으로 클라이언트 사이드 JavaScript가 없습니다
2. **컴포넌트 아일랜드** - 필요한 곳에만 인터랙티브 컴포넌트를 추가합니다
3. **빠른 빌드** - Vite 기반의 빠른 개발 경험을 제공합니다

## 설치

```bash
npm create astro@latest
```

> Astro는 콘텐츠 중심 사이트에 최적화되어 있습니다.
